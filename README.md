# LMao Task Manager

Website quản lý công việc hiện đại được xây dựng bằng Next.js và TypeScript.

## 📌 Giới thiệu

LMao Task Manager là một nền tảng quản lý công việc và cộng tác nhóm hiện đại, được thiết kế nhằm hỗ trợ người dùng quản lý deadline, công việc, lịch trình và tiến độ làm việc hiệu quả hơn.

Dự án lấy cảm hứng từ các nền tảng hiện đại như:

* Notion
* Linear
* Trello
* Jira
* Discord UI
* Các SaaS Dashboard hiện đại

## ⚙️ Hướng dẫn cài đặt

Clone source code:

```bash
git clone https://github.com/Lie-5/Quanlicongviec-mini.git
```

Cài đặt thư viện:

```bash
npm install
```

Chạy project:

```bash
npm run dev
```

Mở trình duyệt:

```bash
http://localhost:3000
```
